# BookStore Library Management System

A WPF desktop application built with C# and SQL Server LocalDB 
that manages library activities for UCOL's BookStore library.

## Features
- Single sign-on login system with two user types (Staff & Student)
- Students can search books, calculate borrow periods and finalise borrows
- Staff can perform full CRUD operations on the book inventory
- Staff can search and filter student borrow records by book or date range
- Sorting functionality by date and price

## Technologies Used
- C# / WPF (.NET 8)
- SQL Server LocalDB
- System.Data.SqlClient
- OOP principles — Inheritance, Encapsulation
- XML Documentation Comments
